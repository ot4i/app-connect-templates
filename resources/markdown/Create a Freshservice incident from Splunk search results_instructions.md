This flow creates a Freshservice incident from Splunk search results.
## Using the template

1. Click **Use this template** to start using the template.
2. Click the **Freshservice** node, and if you're not already connected, connect to your [Freshservice account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-freshservice).
3. Click the **Splunk** node, and if you're not already connected, connect to your [Splunk account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-splunk).
4. To start the flow, in the banner click **Deploy flow**.

Use this template to create a Freshservice incident from Splunk search results.
