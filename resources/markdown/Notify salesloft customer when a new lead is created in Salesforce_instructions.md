This flow notifies a Salesloft customer when a new lead is created in Salesforce.

1. Click **Use this template** to start using the template.
2. Click the **Salesloft** node, and if you're not already connected, connect to your [Salesloft account](http://ibm.biz/acsalesloft).
3. Click the **Salesforce** node, and if you're not already connected, connect to your [Salesforce account](http://ibm.biz/acsalesforce).
5. To start the flow, in the banner, click **Start flow** or **Deploy flow**.

Use this template to notify a Salesloft customer when a new lead is created in Salesforce.
