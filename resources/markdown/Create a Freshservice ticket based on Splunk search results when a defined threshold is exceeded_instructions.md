This flow creates a Freshservice ticket based on Splunk search results when a defined threshold is exceeded.

1. Click **Use this template** to start using the template.
2. Click the **Freshservice** node, and if you're not already connected, connect to your [Freshservice account](http://ibm.biz/acfreshservice).
3. Click the **Splunk** node, and if you're not already connected, connect to your [Splunk account](http://ibm.biz/acsplunk).
4. To start the flow, in the banner, click **Start flow** or **Deploy flow**.

Use this template to create a Freshservice ticket based on Splunk search results when a defined threshold is exceeded.
