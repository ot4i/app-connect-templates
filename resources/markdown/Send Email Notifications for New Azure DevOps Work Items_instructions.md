Use this template to automatically send an email notification through Microsoft Exchange when a new work item is created in Azure DevOps.

To refer to these instructions while editing the flow, open [the GitHub page](https://github.com/ot4i/app-connect-templates/blob/main/resources/markdown/Send%20email%20notifications%20for%20new%20Azure%20DevOps%20work%20items_instructions.md) (opens in a new window).

1. Click **Use this template** to start using the template.
2. Connect to the following accounts by using your credentials:
   - [Azure DevOps](https://ibm.biz/acazuredevops)
   - [Microsoft Exchange](https://ibm.biz/acmsexchange)
3. In the Azure DevOps trigger, update the `organization`, `project`, and `name` fields to match your Azure DevOps environment.
4. In the **Microsoft Exchange Send email** action, update the `toRecipientsRequested` field with the recipient email address.
5. To start the flow, in the banner, click **Start flow**.
