To refer to these instructions while editing the flow, open [the GitHub page](https://github.com/ot4i/app-connect-templates/tree/main/resources/markdown/Updating%20status%20changes%20in%20Salesforce%20opportunity%20to%20Salesloft%20notes_instructions.md) (opens in a new window).

This flow creates a Salesloft note for a corresponding customer when there is a status change in Salesforce oppurtunity.

1. Click **Use this template** to start using the template.
2. Click the **Salesloft** node, and if you're not already connected, connect to your [Salesloft account](http://ibm.biz/acsalesloft).
3. Click the **Salesforce** node, and if you're not already connected, connect to your [Salesforce account](http://ibm.biz/acsalesforce).
5. To start the flow, in the banner, click **Start flow** or **Deploy flow**.

Use this template to create a Salesloft note for a corresponding customer when there is a status change in Salesforce oppurtunity.
