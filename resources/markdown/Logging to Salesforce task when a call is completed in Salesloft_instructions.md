This flow creates a Salesforce task when a call activity is completed in Salesloft.

1. Click **Use this template** to start using the template.
2. Click the **Salesloft** node, and if you're not already connected, connect to your [Salesloft account](http://ibm.biz/acsalesloft).
3. Click the **Salesforce** node, and if you're not already connected, connect to your [Microsoft Teams account](http://ibm.biz/acsalesforce).
5. To start the flow, in the banner, click **Start flow** or **Deploy flow**.

Use this template for logging to Salesforce task when a call activity is completed in Salesloft.
