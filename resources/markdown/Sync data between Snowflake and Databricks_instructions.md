To refer to these instructions while editing the flow, open [the GitHub page](https://github.com/ot4i/app-connect-templates/tree/main/resources/markdown/Sync%20data%20between%20Snowflake%20and%20Databricks_instructions.md) (opens in a new window).

1. Click **Use this template** to start using the template.
2. Connect to the following accounts by using your credentials:
   - [Snowflake](https://ibm.biz//acsnowflake) 
   - [Databricks](https://ibm.biz/acdatabricks) 

3. To start the flow, in the banner click **Start flow**.

Use this template to retrieve rows from a table in Snowflake and use this data to insert rows in bulk to a table in Databricks.
