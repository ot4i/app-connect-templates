This template retrieves a ServiceNow problem, generates a draft resolution using Azure OpenAI, and sends it through Gmail.

To refer to these instructions while editing the flow, open [the GitHub page](https://github.com/ot4i/app-connect-templates/blob/main/resources/markdown/Generate%20a%20draft%20resolution%20for%20a%20ServiceNow%20problem%20using%20Azure%20OpenAI%20and%20send%20it%20through%20Gmail_instructions.md) (opens in a new window).

1. Click **Use this template** to start using the template.
2. Click the **Microsoft Azure OpenAI** node, and if you're not already connected, connect to your [Microsoft Azure OpenAI account](http://ibm.biz/acazureopenai).
3. Click the **Servicenow** node, and if you're not already connected, connect to your [ServiceNow account](http://ibm.biz/acservicenow).
4. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](http://ibm.biz/acgmail).
5. Click the **HTTP** node, and if you're not already connected, connect to your [HTTP account](http://ibm.biz/ach2http).
6. To start the flow, in the banner, click **Start flow**.

Use this template to automate the process of customer service assistant with a draft resolution for the issue being sent to the user.
