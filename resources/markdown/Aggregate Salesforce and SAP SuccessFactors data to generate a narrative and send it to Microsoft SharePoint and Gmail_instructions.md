This template gets the contact data from Salesforce, adds it to SAP SuccessFactors as a candidate, generates a narrative about the candidate with their contact details, and sends it to Gmail and Microsoft Sharepoint. 

To refer to these instructions while editing the flow, open [the GitHub page](https://github.com/ot4i/app-connect-templates/blob/main/resources/markdown/Aggregate%20Salesforce%20and%20SAP%20SuccessFactors%20data%20to%20generate%20a%20narrative%20and%20send%20it%20to%20Microsoft%20SharePoint%20and%20Gmail_instructions
) (opens in a new window).

1. Click **Use this template** to start using the template.
2. Click the **Microsoft Azure OpenAI** nodes, and if you're not already connected, connect to your [Microsoft Azure OpenAI account](http://ibm.biz/acazureopenai).
3. Click the **Salesforce** node, and if you're not already connected, connect to your [Salesforce account](http://ibm.biz/ach2salesforce).
4. Click the **SAP SuccessFactors** node, and if you're not already connected, connect to your [SAP SuccessFactors account](http://ibm.biz/acsapsuccessfactors).
5. Click the **Microsoft SharePoint** node, and if you're not already connected, connect to your [Microsoft Sharepoint account](http://ibm.biz/acsharepoint).
6. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](http://ibm.biz/acgmail).
7. To start the flow, in the banner, click **Start flow**.

Use this template to automate the process of executing summaries aligned with enterprise recruitment data.
