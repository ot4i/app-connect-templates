This template automatically creates business partner records in SAP S/4HANA when new leads are created or updated in Zoho CRM, streamlining the supplier onboarding process and sending a notification through Gmail.

To refer to these instructions while editing the flow, open [the Github page](https://github.com/ot4i/app-connect-templates/blob/main/resources/markdown/Synchronize%20supplier%20onboarding%20data%20from%20Zoho%20CRM%20to%20SAP%20S4HANA%20vendors%20and%20send%20a%20notification%20through%20Gmail_instructions.md) (opens in a new window).

1. Click **Use this template** to start using the template.
2. Click the **Zoho CRM** node, and if you're not already connected, connect to your [Zoho CRM account](https://ibm.biz/aczohocrm).
3. Click the **SAP S/4HANA** node, and if you're not already connected, connect to your [SAP S/4HANA account](https://ibm.biz/acsaps4hana).
4. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](https://ibm.biz/acgmail).
5. To start the flow, in the banner, click **Start flow**.

Use this template to synchronize supplier onboarding data from Zoho CRM to SAP S/4HANA vendors and send a notification through Gmail.
