This flow transfers a file through IBM Aspera and sends the transfer status in Gmail upon approval received in Microsoft Exchange.
## Using the template

1. Click **Use this template** to start using the template.
1. Click the **IBM Aspera** node, and if you're not already connected, connect to your [IBM Aspera account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-aspera).
1. Click the **Microsoft Exchange** node, and if you're not already connected, connect to your [Microsoft Exchange account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-microsoft-exchange).
1. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-gmail).
1. To start the flow, in the banner click **Deploy flow**.

Use this template to transfer a file through IBM Aspera and send the transfer status in Gmail upon approval received in Microsoft Exchange.
