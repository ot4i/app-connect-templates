## Using the template

1. Click **Use this template** to start using the template.
2. Connect to the following accounts by using your credentials:
   - [Hive](https://ibm.biz/achive)
   - [SAP S4HANA](https://ibm.biz/acsaps4hana) 
   - [Salesforce](https://ibm.biz/acgbigquery)
3. To start the flow, in the banner, click **Start flow**.

Use this template to batch retrieve customer numbers from Hive, enrich with SAP S4HANA customer details, and load into BigQuery.
