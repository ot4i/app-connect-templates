This flow transfers a file through IBM Aspera and send the transfer status in MS Exchange upon approval received in MS Exchange.
## Using the template

1. Click **Use this template** to start using the template.
2. Click the **IBM Aspera** node, and if you're not already connected, connect to your [IBM Aspera account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-aspera).
3. Click the **Microsoft Exchange** node, and if you're not already connected, connect to your [Microsoft Exchange account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-microsoft-exchange).
4. To start the flow, in the banner click **Deploy flow**.

Use this template to transfer a file through IBM Aspera and send the transfer status in MS Exchange upon approval received in MS Exchange.
