To refer to these instructions while editing the flow, open [the GitHub page](https://github.com/ot4i/app-connect-templates/tree/main/resources/markdown/Send%20a%20mail%20requesting%20feedback%20when%20a%20Freshservice%20ticket%20is%20resolved_instructions.md) (opens in a new window).

This flow triggers a Salesloft cadence which can be configured to send a mail requesting feedback on Freshservice ticket resolution.

1. Click **Use this template** to start using the template.
2. Click the **Salesloft** node, and if you're not already connected, connect to your [Salesloft account](http://ibm.biz/acsalesloft).
3. Click the **Freshservice** node, and if you're not already connected, connect to your [Freshservice account](http://ibm.biz/acfreshservice).
4. To start the flow, in the banner, click **Start flow** or **Deploy flow**.

Use this template to trigger a Salesloft cadence which can be configured to send a mail requesting feedback on Freshservice ticket resolution.
