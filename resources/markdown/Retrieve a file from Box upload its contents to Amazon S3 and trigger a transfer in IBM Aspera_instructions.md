To refer to these instructions while editing the flow, open [the GitHub page](https://github.com/ot4i/app-connect-templates/tree/main/resources/markdown/Retrieve%20a%20file%20from%20Box%20upload%20its%20contents%20to%20Amazon%20S3%20and%20trigger%20a%20transfer%20in%20IBM%20Aspera_instructions.md) (opens in a new window).

This flow triggers an IBM Aspera transfer upon receiving transfer approval from Microsoft Exchange.

## Using the template

1. Click **Use this template** to start using the template.
1. Click the **IBM Aspera** node, and if you're not already connected, connect to your [IBM Aspera account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-aspera).
1. Click the **Box** node, and if you're not already connected, connect to your [Box account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-box).
1. Click the **Box** node, and if you're not already connected, connect to your [Amazon S3 account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-amazon-s3).
1. To start the flow, in the banner click **Deploy flow**.

Use this template to retrieve a file from Box, upload its contents to Amazon S3, and trigger a transfer in IBM Aspera.
