This template gets campaign details from salesforce and create embeddings using Azure OpenAI and saves in Pinecone DB. 

To refer to these instructions while editing the flow, open [the Github page](https://github.com/ot4i/app-connect-templates/blob/main/resources/markdown/Generate%20content%20similarity%20insights%20by%20creating%20or%20updating%20vectors%20in%20Pinecone%20Vector%20DB%20using%20Azure%20OpenAI%20embeddings_instructions
.md) (opens in a new window).

1. Click **Use this template** to start using the template.
2. Click the **Microsoft Azure OpenAI** nodes, and if you're not already connected, connect to your [Microsoft Azure OpenAI account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-azure-openai).
3. Click the **Pinecone Vector Database** node, and if you're not already connected, connect to your [Pinecone Vector Database account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-pinecone-vector-database).
4. Click the **Salesforce** node, and if you're not already connected, connect to your [Salesforce account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-salesforce).
5. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-gmail).
6. To start the flow, in the banner, click **Start flow**.

Use this template to generate content similarity insights using Azure OpenAI embeddings and save in Pinecone Vector DB.
