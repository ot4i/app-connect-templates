This flow sends a Microsoft Teams message and a Gmail message when a new ticket is created in Freshservice.
## Using the template

1. Click **Use this template** to start using the template.
2. Click the **Freshservice** node, and if you're not already connected, connect to your [Freshservice account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-freshservice).
3. Click the **Microsoft Teams** node, and if you're not already connected, connect to your [Microsoft Teams account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-microsoft-teams).
4. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-gmail).
5. To start the flow, in the banner click **Deploy flow**.

Use this template to send a Microsoft Teams message and a Gmail message when a new ticket is created in Freshservice.
