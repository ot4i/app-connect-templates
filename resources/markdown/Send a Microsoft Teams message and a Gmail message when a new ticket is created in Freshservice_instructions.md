This flow sends a Microsoft Teams message and a Gmail message when a new ticket is created in Freshservice.

1. Click **Use this template** to start using the template.
2. Click the **Freshservice** node, and if you're not already connected, connect to your [Freshservice account](http://ibm.biz/acfreshservice).
3. Click the **Microsoft Teams** node, and if you're not already connected, connect to your [Microsoft Teams account](http://ibm.biz/acmsteams).
4. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](http://ibm.biz/acgmail).
5. To start the flow, in the banner, click **Start flow** or **Deploy flow**.

Use this template to send a Microsoft Teams message and a Gmail message when a new ticket is created in Freshservice.
