This flow uploads a Box file to Amazon S3 and transfer it to a target application through IBM Aspera.
## Using the template

1. Click **Use this template** to start using the template.
1. Click **Operations**, then **Edit flow** for the _Create transfer_ operation.
1. Click the **IBM Aspera** node, and if you're not already connected, connect to your [IBM Aspera account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-aspera).
1. Click the **Box** node, and if you're not already connected, connect to your [Box account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-box).
1. Click the **Amazon S3** node, and if you're not already connected, connect to your [Amazon S3 account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-amazon-s3).
1. To start the flow, in the banner click **Start flow**.

Use this template to upload a Box file to Amazon S3 and transfer it to a target application through IBM Aspera.
