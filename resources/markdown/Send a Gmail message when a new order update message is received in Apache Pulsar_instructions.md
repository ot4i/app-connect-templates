This template is the continuation of “Publish an order update message to an Apache Pulsar topic when a new order is created in Shopify” template, enabling you to send Gmail notifications after the order update message is received in Apache Pulsar.

To refer to these instructions while editing the flow, open [the Github page](https://github.com/ot4i/app-connect-templates/blob/main/resources/markdown/Send%20a%20Gmail%20message%20when%20a%20new%20order%20update%20message%20is%20received%20in%20Apache%20Pulsar_instructions.md) (opens in a new window).

1. Click **Use this template** to start using the template.
2. Click the **Apache Pulsar** node, and if you're not already connected, connect to your [Apache Pulsar account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-apache-pulsar).
3. Click the **Gmail** node, and if you're not already connected, connect to your [Gmail account](https://www.ibm.com/docs/en/app-connect/saas?topic=apps-gmail).
4. To start the flow, in the banner click **Start flow**.

Use this template to send a Gmail message when a new order update message is received in Apache Pulsar.
